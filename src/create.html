私にとって、自分自身について書くことはとても難しいです。今日の私を説明する唯一の方法は、私を形作った背景から始めることです。私はネパールの小さな村で生まれ、15 歳まで住んでいました。15歳から18歳までは勉強する目的でネパールの首都カドマンズで家族と離れて暮らしてきました。そして19歳の時に来日し現在は 四国大学の学生として勉強しています.

自分が育った家はそんなにお金持ちの家ではなかったので子供のごろは楽しんで生活することができませんでした。朝起きて農作業の仕事を手伝って学校へ行ってまた学校から帰ってきてからも農作業を仕事を手伝って毎日生活をしていました。 子供のごろは家にテレビもスマホゲーム機など何もなく初めてスマホを手に入れたのは17歳のごろでした　初めてパソコンを手に入れたのは２１歳の時でした。

ネパールの高校の時は教育の勉強しました。 子供ごろはあまりテクノロジーに関わることができませんでしたが 高校の時からはテクノロジーについて興味を持ちました。高校後の教育はネパールでしても勉強をし終わってからあまり仕事ができる会社がないので高校後日本への留学を決め２０１９年に来日し今は四年間経ちました。

将来の夢は母国で会社を経営することです。大学卒業後は日本のIT企業に就職し多くの経験を積みたいです。現在は大学を通いながらバイトもしています。 最近は将来へ向けて独学でプログラミングを勉強しています。主に使う言語は Python ,Java,JavaScprit ですが一番多いは Python の　フレームワーク Django を使うことが多いです。 Djangoを使って色々なアプリ開発したりしています。最近開発しているアプリは 電子商取引アプリケーションです。今のアプリ開発を終わったらリアルタイムのアプリケーションを作る予定をしています。今後も 夢を叶えるように頑張っていきたいと思っています。
